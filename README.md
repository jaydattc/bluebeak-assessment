# Assessment Test for [bluebeak.ai](https://bluebeak.ai/)

## Run this project

> $ git clone https://github.com/jaydattc/bluebeak-assessment
>
> $ cd bluebeak-assessment
>
> $ yarn install
>
> $ yarn run start

or

> $ git clone https://github.com/jaydattc/bluebeak-assessment &&
> cd bluebeak-assessment &&
> yarn install &&
> yarn run start
